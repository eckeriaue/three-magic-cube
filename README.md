## after assembly do the following
 > rename "dist" on "app"
 > in `index.html` replace
 ```html
 <script type="module" crossorigin src="/assets/index.js"></script>
 ```
 to 
 ```html
 <script type="module" crossorigin src="./assets/index.js"></script>
 ```
